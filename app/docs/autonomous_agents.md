# Autonomous Internet Economic Ecosystem v2

## Safety and ethics
- Automation must not violate platform terms, law, privacy, or anti-fraud controls.
- External actions are auditable and routed via API Gateway.
- High-value transfers/hiring are escalated to Judge review.

## Architecture
- Sensing streams -> sensing bus -> opportunity engine -> business builder.
- Human task market is limited to physical-world tasks.
- Financial operations use tamper-evident hash-chained logs.
